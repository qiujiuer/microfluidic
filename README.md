### README 1: 明场图像微孔状态自动标注工具

# ddPCR 明场图像微孔状态自动标注工具

## 1. 项目简介
本脚本用于批量处理 ddPCR 明场图像，自动识别 20×20 微孔阵列中每个孔的状态。通过分析图像的几何特征和亮度特征，将微孔分类为：**正常液滴**、**伪影/合并** 和 **空/无效**，并输出标注结果与可视化图像。

## 2. 环境依赖
- Python 3.x
- OpenCV (`cv2`)
- NumPy
- Pandas
- scikit-learn (`sklearn`)
- tifffile

## 3. 数据集目录结构
请确保数据集目录结构如下：
```
data\pycharm_file\patchs\dataset_bright\
├── images\            # 存放待处理的 .tif 明场图像
└── labels\            # 输出目录（脚本自动创建）
    ├── geom\          # 微孔几何位置信息
    ├── state\         # 微孔状态标签
    ├── debug\         # 调试特征数据
    ├── vis\           # 可视化叠加图
    └── AE_patches\    # 异常样本切片图
```

## 4. 使用方法
1. **配置路径**：修改脚本中 `dataset_root` 变量为您的数据集根目录。
2. **参数调整**（可选）：
   - `max_images`: 试跑时可设为整数（如 10），正式运行设为 `None`。
   - `save_overlay_every`: 每隔多少张图保存一张可视化。
   - 阈值参数（如 `E_mean_th`, `E_std_th`）可根据实际样本微调。
3. **运行脚本**：
   ```bash
   python biaoqian_baright.py
   ```

## 5. 核心算法逻辑
### 5.1 定位与对齐
- **Hough 圆检测**：检测图像中的微孔位置。
- **网格分配**：利用 KMeans 聚类将检测到的圆心分配至 20×20 的网格中。
- **透视变换**：校正图像倾斜，生成标准的对齐图像。

### 5.2 分类判据
- **空/无效 (Label 2, E)**：
  - 依据：整体亮度均值过低、中心亮度过低、或标准差与边缘比率同时过低。
  - 物理意义：孔内无液滴或成像质量极差。
- **伪影/合并 (Label 1, A)**：
  - 依据：基于距离变换，计算核心连通域数量。若核心数量 ≥ 2，判定为合并。
  - 物理意义：孔内存在多个液滴或杂质。
- **正常液滴 (Label 0)**：
  - 依据：未触发上述异常条件。

## 6. 输出文件说明
- `*_well_geometry.csv`：每个孔的对齐后坐标、间距、物理尺寸换算信息。
- `*_well_labels.csv`：每个孔的状态标签（0/1/2）及备注。
- `*_overlay.png`：可视化结果，蓝色圆圈为正常，白色字母 A/E 标注异常。
- `AE_patches/`：自动导出被判定为 A 或 E 的微孔切片图，便于人工复核。

### README 2: 荧光图像液滴阴阳性自动标注工具 (v4)

# ddPCR 荧光图像液滴阴阳性自动标注工具 (v4)

## 1. 项目简介
本脚本（v4版本）专门用于处理 ddPCR 荧光图像，自动分割液滴并分类为 **阴性**、**阳性** 或 **气泡**。v4 版本核心改进在于引入 **SNR_center (信噪比中心特征)**，解决了不同批次图像亮度差异大导致的阈值失效问题。

## 2. 环境依赖
- Python 3.x
- OpenCV (`cv2`)
- NumPy
- Pandas
- scikit-learn (`sklearn.cluster`, `sklearn.mixture`)
- SciPy (`scipy.stats`)
- Pillow (`PIL`)

## 3. 数据集目录结构
```
data\pycharm_file\patchs\dataset_dark_example\
├── images\            # 明场图（用于辅助定位微孔网格）
├── images_fluor\      # 荧光图（待处理的主图像）
└── labels_v4\         # 输出目录
    ├── state\         # 阴阳性标签
    ├── debug\         # SNR 等特征详情
    ├── vis\           # 叠加可视化图
    └── pos_patches\   # 阳性液滴切片
```

## 4. 使用方法
1. **配置路径**：修改 `dataset_root` 指向您的数据集。
2. **运行脚本**：
   ```bash
   python biaoqian_dark.py
   ```
3. **模式选择**：
   - 默认运行批量处理 `run_batch()`。
   - 可调用 `test_single(path)` 对单张图片进行调试。

## 5. 核心算法逻辑 (v4)
### 5.1 特征提取：SNR_center
- **定义**：`SNR_center = 液滴中心40%半径区域均值 / 液滴外圈背景均值`
- **优势**：比值特征，自动归一化，不受曝光强弱、背景亮度波动影响。
- **物理依据**：
  - 阴性：透镜效应，中心暗，SNR ≈ 0.7~1.2。
  - 阳性：荧光充满，中心亮，SNR > 1.5。

### 5.2 三级自适应阈值策略
脚本不使用固定阈值，而是根据每张图的数据分布自动计算：
1. **GMM 模型**：若阴阳性双峰分离清晰（sep ≥ 3.5），取高斯交叉点。
2. **阴性锚定法（首选）**：取图像中最暗的 15% 液滴作为阴性样本，计算 `neg_μ + 2.8σ` 作为阈值。
3. **固定兜底**：若上述方法失效，使用默认值 1.20。

### 5.3 气泡检测
- 若液滴区域最大灰度值 `gmax` 接近相机噪底 (`noise_floor`)，判定为气泡。

## 6. 输出文件说明
- `*_fluor_labels.csv`：包含 `label_id` (0:阴性, 1:阳性, 2:气泡) 及计算得到的 SNR 值。
- `*_fluor_debug.csv`：详细的中心均值、环均值、背景均值等特征，用于算法调优。
- `*_fluor_overlay.png`：可视化图，绿色=阴性，黄色+P=阳性，红色+X=气泡。图片底部包含统计信息栏。
- `fluor_batch_summary.csv`：整批处理结果的统计汇总。
